# AI Supply Chain Predictor

This is a machine learning prototype designed to help businesses forecast their daily sales. By analyzing historical data, this tool predicts future demand to assist with inventory planning and supply chain management. 

Currently, this is a prototype using dummy data, but it effectively demonstrates how machine learning can drive intelligent business decision-making.

##  Tech Stack
* **Python:** The core programming language.
* **Scikit-Learn:** Used for the Linear Regression machine learning model.
* **Streamlit:** Used to build the interactive web frontend.
